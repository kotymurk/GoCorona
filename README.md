GoCorona — Современный адаптивный лендинг

Чистая и профессиональная верстка одностраничного сайта, посвященного медицинской тематике. Проект демонстрирует навыки создания интерфейсов с упором на доступность и адаптивность под любые устройства.

## 🔗 Ссылки
- Live Demo: https://go-corona-gamma.vercel.app
- GitHub: [https://github.com/kotymurk/GoCorona](https://github.com/kotymurk/GoCorona)

## 🛠 Технологии
- HTML5 (Семантическая верстка для лучшего SEO)
- CSS3 / SCSS (Использование переменных, миксинов и вложенности)
- BEM Methodology — строгая структура классов для поддержки кода
- Flexbox & CSS Grid — создание гибких и надежных сеток
- JavaScript (Vanilla) — легкая интерактивность и мобильное меню

## 🚀 Особенности проекта
- Mobile First: Разработка велась от мобильной версии к десктопной, что гарантирует идеальное отображение на смартфонах.
- Pixel Perfect: Максимальное соответствие исходному дизайн-макету.
- Оптимизация: Все изображения сжаты, а шрифты подключены максимально производительно.
- Анимации: Плавные переходы и hover-эффекты для улучшения пользовательского опыта (UX).

## 📂 Структура проекта
- index.html — основная разметка
- scss/ или css/ — стили проекта (разбиты по компонентам БЭМ)
- img/ — оптимизированная графика
- js/ — скрипты для работы меню и других элементов
